# 混合创作模式

## 推荐模式：AI生成 + 人工改写

### 比例建议
- AI生成：40-60%
- 人工改写：40-60%
- 识别率：双盲测试仅29%（vs 纯AI 83%）

### 操作流程

#### 第一步：AI生成初稿
- 使用AI生成章节初稿
- 提供详细的写作指令
- 包含人设、剧情、风格要求

#### 第二步：标记AI痕迹
- 通读初稿
- 标记AI痕迹明显的段落
- 标记需要改写的部分

#### 第三步：深度改写
- 对标记段落进行深度改写
- 注入口语化表达
- 添加个人风格特征
- 调整句式和节奏

#### 第四步：全文统一
- 通读全文确保风格统一
- 检查人设一致性
- 调整节奏和情绪

#### 第五步：最终检查
- 检查AIGC特征
- 确保无敏感内容
- 确认平台合规

## 其他模式

### AI辅助构思+人工写作
- AI提供创意和框架
- 人工完成全部写作
- 识别率：极低

### 人工写作+AI润色
- 人工完成初稿
- AI辅助润色修改
- 识别率：低

### 纯人工写作
- 不使用AI
- 识别率：零（但效率最低）
