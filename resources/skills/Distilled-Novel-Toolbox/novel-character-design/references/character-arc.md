# 人物弧光设计方法

## 正面弧光（最常用）

### 结构
1. 起始状态：角色有明显缺陷或不足
2. 触发事件：遇到无法用旧方式解决的问题
3. 挣扎抗拒：试图用旧方式解决，失败
4. 转折点：意识到必须改变
5. 成长蜕变：克服缺陷，获得新力量/认知
6. 最终考验：用新自我解决终极挑战

### 适用场景
- 废柴逆袭、成长向、热血向
- 主角需要明显成长轨迹

## 负面弧光

### 结构
1. 起始状态：角色有优点但有小缺陷
2. 诱惑出现：权力/复仇/执念的诱惑
3. 妥协堕落：一步步做出错误选择
4. 越陷越深：无法回头
5. 毁灭/悲剧：最终走向毁灭

### 适用场景
- 反派塑造、悲剧向、警示向
- 需要谨慎使用，避免读者不适

## 平坦弧光

### 结构
1. 起始状态：角色有坚定信念
2. 外部考验：世界试图改变角色
3. 坚持信念：角色坚持不改初心
4. 影响他人：角色的坚持改变了周围人
5. 最终胜利：信念获得验证

### 适用场景
- 理想主义主角、信念坚定的角色
- 适合已经成熟的主角

## 弧光设计要点

1. 弧光必须与剧情主线紧密结合
2. 每个转折点都需要有触发事件
3. 成长不能一蹴而就，需要反复挣扎
4. 弧光的终点要呼应起点
5. 配角也需要弧光，只是幅度可以小一些
