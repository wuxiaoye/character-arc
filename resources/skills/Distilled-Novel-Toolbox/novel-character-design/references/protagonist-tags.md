# 主角性格标签详解

## 标签组合原则

1. 核心标签1-2个（决定主角基本性格）
2. 辅助标签1-2个（增加层次感）
3. 隐藏标签0-1个（反差/成长空间）
4. 标签间不能矛盾（除非刻意设计双面人格）

## 热门标签组合

| 组合 | 效果 | 适用题材 |
|------|------|---------|
| 杀伐果断+谨慎稳健 | 该出手时绝不犹豫，不该出手时稳如老狗 | 玄幻/仙侠 |
| 逗比+智商碾压 | 表面搞笑实则深谋远虑 | 都市/玄幻 |
| 高冷+反差萌 | 偶尔破防的冰山 | 言情/都市 |
| 病娇+温润 | 温柔外表下的偏执 | 言情/耽美 |
| 老阴比+杀伐果断 | 算计完再动手 | 仙侠/权谋 |
| 洒脱+实力超群 | 不争而自得 | 仙侠/武侠 |

## 标签深度开发

每个标签不只是表面特征，需要深入到：
- 行为模式：这个标签如何影响日常行为
- 决策逻辑：这个标签如何影响关键抉择
- 人际关系：这个标签如何影响与他人的互动
- 成长空间：这个标签有哪些可以发展/改变的方向
