# 配角功能模型详解

## 配角设计原则

1. 每个配角必须有明确功能（推动剧情/衬托主角/提供信息/情感支撑）
2. 配角有自己的生活和故事，不只是主角的附属
3. 配角数量控制在可驾驭范围内
4. 重要配角需要有弧光

## 功能模型详解

### 忠诚护卫型
- 功能：战斗辅助、情感羁绊、牺牲高光
- 性格特征：忠诚、直率、重义
- 弧光方向：从跟随者到独立战士
- 经典范例：各种主角的兄弟/护卫

### 军师谋士型
- 功能：策略规划、信息分析、剧情推动
- 性格特征：聪明、谨慎、有时过于算计
- 弧光方向：从算计到真心
- 经典范例：诸葛亮式谋士

### 亦敌亦友型
- 功能：实力参照、互相促进、戏剧张力
- 性格特征：骄傲、有原则、实力强
- 弧光方向：从对手到盟友
- 经典范例：佐助式对手

### 红颜知己型
- 功能：情感支撑、剧情催化剂、读者代入
- 性格特征：温柔/活泼/冷艳，与主角互补
- 弧光方向：从旁观到并肩
- 注意：避免工具人化，给独立故事线
