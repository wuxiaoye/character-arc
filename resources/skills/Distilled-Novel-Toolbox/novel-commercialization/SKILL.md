# 商业化与平台特化

## Purpose

提供网文商业化全链路指南，帮助作者选择最适合的平台，了解各平台风格与规则，设计最优变现策略。

## When to Use

- 用户需要选择发布平台
- 用户需要了解某平台的风格与规则
- 用户需要设计变现模式
- 用户需要规划签约与收入策略

## Instructions

### 平台全景对比

#### 起点中文网
- 定位：最大的付费阅读平台
- 读者画像：男性为主，18-35岁，付费意愿强
- 适合题材：玄幻、仙侠、都市、科幻、历史
- 风格要求：有一定文学性，剧情有深度
- 签约门槛：中等，需1-3万字内投
- 收入模式：订阅+打赏+月票+道具
- 优势：品牌效应强、IP开发渠道多、创作体系完善
- 劣势：竞争激烈、新人出头难

#### 番茄小说
- 定位：免费阅读平台（背靠抖音/字节）
- 读者画像：泛读者，年龄跨度大，免费阅读
- 适合题材：爽文、短平快、都市、言情
- 风格要求：节奏快、爽感密、信息直给
- 签约门槛：较低
- 收入模式：广告分成+全勤（600元/月）
- 优势：流量优势明显、入行门槛低
- 劣势：收入不透明、单本收入上限低

#### 飞卢小说网
- 定位：脑洞/同人/快节奏平台
- 读者画像：年轻男性，追求爽感
- 适合题材：同人、诸天、脑洞、系统流
- 风格要求：爽感十足、脑洞大、节奏极快
- 签约门槛：低（8千字可内投）
- 收入模式：订阅+打赏
- 优势：门槛低、更新快、变现快
- 劣势：更新压力极大、作品寿命短

#### 晋江文学城
- 定位：女性向阅读平台
- 读者画像：女性占比90%+，18-30岁
- 适合题材：言情、耽美、大女主、古言
- 风格要求：情感细腻、文笔好、人设精致
- 签约门槛：中等
- 收入模式：订阅+打赏+定制
- 优势：女性向第一平台、IP价值高
- 劣势：男性向题材不吃香、审核严格

#### 七猫小说
- 定位：免费阅读平台
- 读者画像：下沉市场，年龄偏大
- 适合题材：赘婿、神医、兵王、都市
- 风格要求：传统爽文、节奏快
- 收入模式：广告分成+全勤
- 优势：下沉市场流量大
- 劣势：题材受限、收入上限低

### 付费 vs 免费核心区别

| 维度 | 付费平台 | 免费平台 |
|------|---------|---------|
| 收入透明度 | 高（卖出多少清楚可算） | 低（广告分成难以验证） |
| 收入上限 | 高（爆款月入百万） | 低（多数几千到几万） |
| 读者质量 | 高（付费=认真读） | 低（泛读者，跳出率高） |
| 写作要求 | 高（需有深度和逻辑） | 低（爽就完事） |
| 竞争强度 | 高 | 中 |
| 适合新人 | 有写作基础的新人 | 零基础新人 |

### 变现设计

1. **订阅收入**：核心收入来源
2. **打赏收入**：粉丝经济
3. **月票/推荐票**：排名与曝光
4. **全勤奖**：稳定更新奖励
5. **IP改编**：影视/动漫/游戏
6. **版权运营**：出版/海外/衍生
