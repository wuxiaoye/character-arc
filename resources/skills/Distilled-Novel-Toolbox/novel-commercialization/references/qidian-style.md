# 起点风格详解

## 平台特征

- 最大的付费阅读平台，品牌效应最强
- 读者付费意愿强，对质量要求高
- 完善的创作体系和晋升机制
- IP开发渠道最广

## 适合题材

- 玄幻/仙侠：核心品类，读者基数最大
- 都市：第二大品类
- 科幻/历史：小众但精品
- 悬疑/灵异：有固定读者群

## 风格要求

- 有一定文学性，不能太白
- 剧情有深度，不能纯爽
- 世界观有厚度，不能太薄
- 人物有弧光，不能太扁

## 签约与推荐

### 签约流程
1. 内投：1-3万字+大纲发给编辑
2. 审核：编辑评估题材/文笔/剧情
3. 签约：A签/B签/分成签
4. 上架：达到字数/收藏要求后上架收费

### 推荐机制
- 编辑推荐：编辑根据质量安排推荐位
- 数据推荐：追读/收藏/订阅数据影响推荐
- 读者推荐：月票/推荐票影响排名
- 算法推荐：根据读者偏好推送
