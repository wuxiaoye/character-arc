# 题材与子类融合引擎

## Purpose

提供网文全题材分类体系、各题材写作模板、市场数据、读者画像，以及题材混合融合的相容性矩阵，帮助作者精准选择题材并创新融合。

## When to Use

- 用户需要选择或确定写作题材
- 用户想了解某题材的市场表现、代表作品、写作套路
- 用户想尝试题材融合（如玄幻+科幻、仙侠+都市）
- 用户需要某题材的写作模板与设定框架

## Instructions

### 题材全景分类

#### 东方幻想系
- **玄幻**：东方玄幻（斗气/灵力体系）、异界大陆（多种族多文明）、高武世界（国术/武道极致）
- **仙侠**：修真流（炼气→大乘飞升）、洪荒流（先天灵宝/圣人证道）、现代修仙（都市+修真融合）
- **武侠**：传统武侠（江湖恩怨/门派争锋）、新派武侠（古龙式写意/温瑞安式奇诡）、国术流（真气/内劲/化境）

#### 现实都市系
- **都市生活**：职场商战、日常温馨、社会百态
- **都市异能**：隐世修真、超能觉醒、异能组织
- **都市重生**：商业帝国、人生重来、先知优势
- **赘婿/兵王/神医**：隐藏身份、王者归来、绝世技艺

#### 言情女性向
- **现代言情**：甜宠、虐恋、豪门、职场
- **古代言情**：宫斗、宅斗、女尊、穿越
- **纯爱/耽美**：校园、都市、古风、奇幻
- **女强/大女主**：独立成长、权谋天下、无CP

#### 历史军事系
- **架空历史**：平行世界、王朝更迭
- **穿越历史**：现代知识+古代舞台
- **考据历史**：正史为骨、细节还原
- **争霸天下**：战争谋略、帝国崛起

#### 科幻未来系
- **硬科幻**：物理定律、技术推演
- **软科幻**：社会实验、哲学思辨
- **末世/废土**：文明崩塌、生存法则
- **星际/太空歌剧**：银河帝国、星际战争
- **赛博朋克**：黑客/义体/巨型企业

#### 悬疑灵异系
- **探案推理**：逻辑链、反转、真相揭露
- **盗墓探险**：机关陷阱、古墓秘辛
- **规则怪谈**：诡异规则、生存指南
- **诡异流**：不可名状、理智侵蚀

#### 其他热门题材
- **游戏/电竞**：虚拟现实、竞技对战
- **奇幻/西幻**：魔法体系、种族设定、克苏鲁
- **无限流/诸天**：任务世界、副本攻略
- **快穿**：多世界穿越、任务完成
- **种田/基建**：领地发展、资源管理
- **灵气复苏/全民觉醒**：现代+超能力觉醒
- **系统流**：金手指系统、任务奖励
- **文娱/文抄公**：娱乐圈、搬运经典
- **学霸/玄学**：知识碾压、天师捉鬼

### 题材选择决策树

1. 确定目标平台（起点→玄幻/仙侠/都市；番茄→爽文/短平快；晋江→言情/耽美；飞卢→脑洞/同人）
2. 确定目标读者（男性18-35→玄幻/都市/科幻；女性18-30→言情/耽美/大女主）
3. 评估自身储备（知识储备→考据类；想象力→奇幻类；生活经验→都市类）
4. 市场趋势研判（搜索当前热门题材与新兴趋势）
5. 差异化定位（在热门题材中找冷门切入点，或冷门题材中做精品）

### 题材融合操作

1. 查阅 `references/genre-fusion-matrix.md` 获取融合相容性
2. 选择主题材（占70%框架）与副题材（占30%调味）
3. 确定融合接口：力量体系、世界观、角色身份、剧情事件
4. 验证融合自洽性：两套设定是否逻辑兼容
5. 参考成功案例：诡秘之主（克苏鲁+蒸汽朋克+升级流）
