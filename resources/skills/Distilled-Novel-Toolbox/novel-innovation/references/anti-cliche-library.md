# 反套路设计库

## 使用原则

1. 反套路不是为了反套路而反套路
2. 反套路必须服务于更好的阅读体验
3. 反套路需要读者对原套路有认知才能生效
4. 过度反套路会让读者无所适从

## 叙事反套路详解

### 反派视角
- 效果：颠覆读者对正邪的认知
- 难度：高，需要让读者共情反派
- 适用：灰色题材、深度向

### 不可靠叙述者
- 效果：读者发现叙述者在说谎时的震撼
- 难度：极高，需要精心设计
- 适用：悬疑、心理

## 设定反套路详解

### 系统是敌人
- 效果：金手指变成最大威胁
- 难度：中，需要合理设计系统目的
- 适用：系统流创新

### 升级是陷阱
- 效果：越强越危险，颠覆升级爽感
- 难度：高，需要平衡爽感与危机
- 适用：修仙/玄幻创新
