# 语言与文风光谱

## Purpose

提供网文语言风格的完整光谱与调控方法，帮助作者选择适合题材与平台的文风，并掌握修辞技法、玩梗体系与画面感技术。

## When to Use

- 用户需要确定或调整文风
- 用户需要选择叙述视角
- 用户需要增强画面感/代入感
- 用户需要植入玩梗/热梗
- 用户需要运用特定修辞技法

## Instructions

### 文风类型光谱

#### 白描风
- 特征：简洁利落、不加修饰、以少胜多
- 适用：武侠、硬汉、冷峻风格
- 示例："他拔刀。刀光一闪。人倒。"
- 量化：句均10字以下，修辞密度<5%

#### 华丽风
- 特征：辞藻丰富、意象密集、画面感强
- 适用：仙侠、奇幻、宫廷
- 示例："九天玄雷劈落，紫电如龙蛇狂舞，将那万丈灵山劈作齑粉。"
- 量化：句均20字以上，修辞密度>30%

#### 口语化
- 特征：贴近日常、生动活泼、接地气
- 适用：都市、搞笑、新媒体文
- 示例："靠，这什么鬼？老子辛辛苦苦修炼三年，就这？"
- 量化：口语填充词>10%，短句为主

#### 文艺风
- 特征：意境深远、留白多、情感细腻
- 适用：言情、文艺、治愈
- 示例："风吹过旧时光的缝隙，带走了那年夏天的蝉鸣。"
- 量化：长句为主，意象密度高

#### 中二风
- 特征：夸张、热血、自我设定感强
- 适用：轻小说、热血向
- 示例："吾之右手封印着黑炎龙，一旦解放，世界将为之颤抖！"
- 量化：感叹句多、专有名词多

#### 翻译腔
- 特征：西式句法、长定语、被动句
- 适用：西幻、科幻
- 示例："那个被命运选中的少年，在黎明到来之前，终于做出了决定。"
- 量化：被动句>15%，长定语多

#### 新媒体风
- 特征：短句、快节奏、信息密集、爽感直给
- 适用：番茄/飞卢/新媒体文
- 示例："叮！签到成功！恭喜获得万年灵药！萧尘嘴角微扬。"
- 量化：句均8-12字，段落<3句

### 叙述视角选择

| 视角 | 优势 | 劣势 | 适用 |
|------|------|------|------|
| 第一人称 | 代入感强、内心丰富 | 视角受限、信息量少 | 悬疑、都市 |
| 第三人称有限 | 平衡代入与视角 | 需控制视角切换 | 大多数网文 |
| 第三人称全知 | 信息量大、灵活 | 代入感弱 | 群像、史诗 |
| 第二人称 | 极强代入 | 难以持续 | 实验性、短篇 |

### 画面感技术

1. **特写**：聚焦关键细节，放大情感冲击
2. **慢镜头**：关键时刻放慢节奏，逐帧描写
3. **蒙太奇**：快速切换场景/时间，制造节奏感
4. **空镜头**：环境/氛围描写，情绪铺垫
5. **长镜头**：一镜到底的连贯动作描写

### 修辞技法库

- 比喻/明喻/暗喻：将抽象具象化
- 拟人：赋予非人以人的特征
- 夸张：放大特征制造效果
- 排比：增强气势与节奏
- 对偶：对称美感
- 通感：跨感官描写
- 留白：不说透，留给读者想象

### 玩梗体系

1. **网文梗**：退婚流、三十年河东、此子断不可留
2. **网络梗**：当前流行语、表情包语言
3. **时事梗**：近期热点事件的巧妙植入
4. **跨圈梗**：其他领域的梗跨界使用
5. **自创梗**：书中反复出现的独特表达
