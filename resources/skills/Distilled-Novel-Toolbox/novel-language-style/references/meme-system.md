# 玩梗体系

## 梗的分类

### 网文经典梗
- "三十年河东，三十年河西"
- "此子断不可留"
- "我有一剑"
- "退婚"
- "小辈，你可知你惹了谁"

### 网络流行梗
- 需要及时更新，过时的梗不如不用
- 用法：角色对话中自然带出
- 注意：不要硬塞，要符合角色性格

### 时事热点梗
- 时效性强，需要快速植入
- 适合新媒体文/都市文
- 注意：避免敏感话题

### 跨圈梗
- 从其他领域引入（游戏/动漫/科技/体育）
- 增加趣味性和辨识度
- 适合轻小说/同人

### 自创梗
- 书中反复出现的独特表达
- 形成作品的标志性语言
- 读者会自发传播

## 植入原则

1. 自然融入，不要突兀
2. 符合角色性格和场景
3. 不要过度使用，偶尔点缀
4. 过时的梗不如不用
5. 敏感梗绝对不用
