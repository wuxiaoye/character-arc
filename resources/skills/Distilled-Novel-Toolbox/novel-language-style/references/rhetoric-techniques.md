# 修辞技法库

## 网文高频修辞

### 比喻
- 功能：将抽象具象化，增强理解与画面感
- 网文用法：力量描写、战斗场景、境界感悟
- 示例："灵力如潮水般涌入经脉"

### 夸张
- 功能：放大特征，制造冲击力
- 网文用法：实力展示、打脸效果、场景描写
- 示例："一拳轰出，万里山河尽碎"

### 排比
- 功能：增强气势，制造节奏感
- 网文用法：升级描写、功法展示、气势渲染
- 示例："他有一剑，可斩天，可裂地，可碎星辰"

### 通感
- 功能：跨感官描写，增强沉浸感
- 网文用法：修炼感悟、环境描写、情绪表达
- 示例："那道声音如同冰水浇在脊背上"

### 留白
- 功能：不说透，留给读者想象空间
- 网文用法：悬念、情感、结局
- 示例："他看着远方的天际，久久无言。"

## 修辞使用原则

1. 服务于内容，不为修辞而修辞
2. 每段最多1-2个修辞，过多则腻
3. 修辞类型要轮换
4. 关键时刻用重修辞，日常用轻修辞
5. 口语化文风少用书面修辞
