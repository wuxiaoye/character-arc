# 高潮编排详解

## 高潮密度控制

### 小高潮（每2-3章）
- 规模：个人级冲突
- 效果：维持阅读兴趣
- 类型：小打脸、小突破、小揭秘

### 中高潮（每8-10章）
- 规模：势力级冲突
- 效果：阶段性满足感
- 类型：大打脸、境界突破、势力对抗

### 大高潮（每卷/30-50章）
- 规模：世界级冲突
- 效果：卷末高潮、读者记忆点
- 类型：终极对决、真相揭露、世界变革

## 高潮结构

### 标准高潮结构
1. 铺垫蓄力（40%）：矛盾积累、压力递增
2. 爆发点（20%）：冲突达到顶点
3. 高潮对决（25%）：正面对决/真相揭露
4. 余波收束（15%）：结果、影响、新平衡

### 高潮后回落

高潮后必须有回落期，不能一直紧绷：
- 回落期长度：高潮长度的1/3-1/2
- 回落期内容：消化结果、角色互动、新信息铺垫
- 回落期功能：让读者喘息、为下一高潮蓄力

## 高潮编排禁忌

1. 高潮过于密集：读者疲劳
2. 高潮之间无铺垫：缺乏期待感
3. 高潮力度递减：越往后越弱
4. 高潮后无回落：一直紧绷
5. 高潮结构单一：每次都是同一种模式
