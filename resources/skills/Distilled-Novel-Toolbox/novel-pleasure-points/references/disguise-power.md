# 扮猪吃虎与隐藏实力类

## 扮猪吃虎

### 心理机制
读者知道主角真实实力而对手不知，形成信息差期待。每次被轻视都在积累期待，展露实力时获得巨大释放。

### 结构模板
1. **隐藏期**：主角因某种原因隐藏实力（低调/任务/试探）
2. **被轻视期**：对手因表面实力而轻视/挑衅/嘲讽
3. **忍耐积累**：主角继续隐忍，读者期待值攀升
4. **爆发点**：触及底线/关键时刻/不再需要隐藏
5. **实力展露**：一击制胜/全面碾压/震惊全场
6. **反应收割**：众人震惊/对手恐惧/旁观者议论

### 量化参数
- 隐藏→爆发比例：3:1（铺垫3份，爆发1份）
- 被轻视次数：2-3次最佳，过多则拖沓
- 爆发力度：与被轻视程度成正比

### 经典范例
- 萧炎初入迦南学院隐藏实力
- 石昊在石村隐藏身份
- 各种"大佬装萌新"桥段

## 越级反杀

### 心理机制
以弱胜强打破等级壁垒，满足"不可能变可能"的幻想。关键在于让读者感受到差距的绝望，再通过智慧/毅力/特殊能力实现逆转。

### 结构模板
1. **实力差距展示**：对手远强于主角
2. **绝境压迫**：主角被逼到极限
3. **突破/智取**：临场突破或巧妙策略
4. **逆转胜利**：以弱胜强实现
5. **代价与成长**：胜利有代价，但获得成长

### 关键技巧
- 差距必须真实可感，不能虚假
- 逆转必须有合理依据（伏笔/特殊能力/策略）
- 胜利可以有代价，增加真实感
