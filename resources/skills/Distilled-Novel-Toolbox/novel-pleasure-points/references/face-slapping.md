# 装逼打脸与逆袭类

## 装逼打脸

### 核心公式
嚣张反派/质疑者 → 主角淡定/轻描淡写 → 实力证明 → 对方崩溃/震惊

### 打脸层次递进
1. **初级打脸**：言语反驳+小范围证明
2. **中级打脸**：行动证明+当众碾压
3. **高级打脸**：不战而屈人之兵+身份揭露
4. **终极打脸**：降维打击+对方世界观崩塌

### 打脸节奏控制
- 单次打脸不超过3个回合，否则拖沓
- 打脸后立即收割反应，不要急着进入下一段
- 反应描写要有层次：当事人→旁观者→更高层

## 人前显圣

### 核心机制
在众人面前展现超凡实力/身份/成就，获得崇拜与敬畏。

### 经典场景
- 拍卖会上豪掷千金
- 比武场上一招制敌
- 危机时刻力挽狂澜
- 身份揭露众人跪拜

## 考场打脸

### 结构模板
1. 被质疑能力（考试/考核/比试）
2. 对方嚣张自信
3. 主角轻松碾压/满分/第一
4. 考官/众人震惊

### 适用题材
学霸文、修仙考核、武道比试、职场竞争
