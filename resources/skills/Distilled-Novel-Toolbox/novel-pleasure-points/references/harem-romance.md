# 后宫与情感关系类

## 后宫/修罗场

### 核心机制
多位异性角色对主角产生好感，形成竞争关系。读者享受被多人喜爱的满足感。

### 角色设计原则
1. 每位后宫角色性格鲜明、不重复
2. 感情发展有逻辑、不突兀
3. 角色间有互动、不只是围绕主角
4. 避免工具人化，每个角色有自己的故事

### 修罗场写法
- 两位及以上角色同时争夺主角关注
- 醋意与竞争制造戏剧张力
- 主角的应对展现性格
- 不能让任何一方完全胜出，保持平衡

## 万人迷/团宠

### 核心机制
主角被众人喜爱、宠溺、保护。满足被爱、被重视的心理需求。

### 类型
- 实力型万人迷：因强大而被崇拜
- 性格型万人迷：因温柔/可爱/有趣而被喜爱
- 身份型万人迷：因身份/血统而被重视
- 综合型万人迷：多方面魅力叠加

## 神豪流

### 核心机制
获得巨额财富，通过花钱获得爽感。

### 结构模板
1. 获得财富来源（系统/遗产/能力）
2. 初始小额消费体验爽感
3. 消费规模逐步升级
4. 用钱解决问题/打脸
5. 财富带来的社会地位变化
