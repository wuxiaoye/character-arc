# 虐渣复仇类

## 虐渣复仇

### 心理机制
正义感+报复快感。读者代入主角，对不公产生愤怒，复仇时获得情绪释放。

### 结构模板
1. **受害期**：主角被伤害/背叛/欺压
2. **积蓄期**：主角默默积蓄力量
3. **复仇期**：逐一清算仇人
4. **高潮期**：最终大仇得报

### 写作要点
- 受害期的痛苦要真实可感，引发读者愤怒
- 复仇要有节奏，不能一蹴而就
- 每个仇人的下场要与其罪行匹配
- 复仇方式要有创意，不只是简单暴力

## 赘婿逆袭

### 标准流程
1. 入赘受辱（妻子/岳家/外人轻视）
2. 隐忍不发（暗中展露冰山一角）
3. 身份揭露（真实身份震惊全场）
4. 全面碾压（曾经轻视者跪求原谅）

## 退婚流

### 标准流程
1. 被退婚（对方嫌贫爱富/嫌弱慕强）
2. 立下誓约（三十年河东三十年河西）
3. 崛起成长（实力飞速提升）
4. 对方后悔（求复合被拒绝）

## 废柴逆袭

### 标准流程
1. 天赋被夺/天生废柴
2. 奇遇获得金手指
3. 逐步崛起打脸
4. 登顶巅峰
