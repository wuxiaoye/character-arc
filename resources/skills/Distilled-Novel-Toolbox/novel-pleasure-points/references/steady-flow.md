# 稳健流与苟道类

## 稳健流/苟道

### 心理机制
在危险世界中追求安全感，通过谨慎和积累实现稳健成长。满足"不冒险也能变强"的心理需求。

### 核心原则
1. 不出头、不冒险、不贪功
2. 默默积累、厚积薄发
3. 关键时刻才展露实力
4. 永远留有后手

### 结构模板
1. 危险世界/环境设定
2. 主角选择低调路线
3. 他人冒险失败→主角稳健获益
4. 被迫出手时一鸣惊人
5. 继续回归低调

### 读者爽点
- 别人冒险主角捡漏
- 危机来临时主角早有准备
- 低调被打破时的实力展现

## 签到/躺赢

### 心理机制
不劳而获的快感，轻松变强的满足。适合快节奏时代的懒人爽感。

### 系统设计
- 签到地点：每到一个新地点签到获奖励
- 签到时间：每日/每月/每年签到
- 签到条件：特定条件触发签到
- 奖励递进：签到越久奖励越丰厚

## 归隐流

### 核心机制
巅峰强者归隐平凡生活，但麻烦不断找上门。

### 结构模板
1. 巅峰时期回忆（简述）
2. 归隐后的平凡生活
3. 麻烦找上门
4. 被迫出手
5. 展露冰山一角
6. 继续归隐/被迫全面出山
