# 润色与去AI味

## Purpose

提供AI生成文本的润色与去AI化方法，帮助作者将AI辅助生成的内容转化为具有人性温度和个人风格的文字。

## When to Use

- 用户需要润色AI生成的网文内容
- 用户需要去除文本中的AIGC痕迹
- 用户需要改写机械句式
- 用户需要增加口语化与个性化表达
- 用户需要注入个人写作风格

## Instructions

### 去AI化核心策略

#### 1. 改写机械句式
- 减少"首先…其次…最后"等完美句式
- 打破"总-分-总"的机械结构
- 避免每段都是"主题句+展开+总结"
- 用不规则句式替代对称句式

#### 2. 添加口语化表达
- 适当使用语气词："靠""哎""真是的""得了吧"
- 内心独白用口语而非书面语
- 对话中增加口头禅和语气词
- 掺入5%口语填充词（比如、其实、不过、话说回来）

#### 3. 融入个人轶事和观点
- 加入主角独特的个人经历回忆
- 插入主观评价和偏见
- 增加不完美但真实的思考过程
- 用个人化的比喻替代通用比喻

#### 4. 调整句子长度变化
- 长短句交替，避免均质
- 关键时刻用短句增强冲击力
- 描写时用长句增加画面感
- 句长变异系数保持在0.3-0.5

#### 5. 增加修辞手法使用
- 用比喻替代直白描述
- 用通感增加感官层次
- 用留白替代过度解释
- 修辞密度控制在15-25%

#### 6. 引入不规则表达
- 偶尔打破语法规则（口语中合理）
- 使用不完整句子
- 插入突然的思路跳跃
- 增加自我纠正和犹豫

### 润色指令模板

**通用去AI化指令**：
```
减少过于完美的句式结构，添加适当不规则表达，使用具个性化语言，偶尔用口语化表达，避免机械化段落结构，保留核心信息和意图，使用2-3成复合句，掺入5%口语填充词（比如、其实、不过等）
```

**战斗场景润色**：
```
增强动作描写的画面感，使用短句增强节奏，添加感官细节（触觉/听觉/嗅觉），减少过度解释，让动作本身说话，加入主角的即时感受和判断
```

**情感场景润色**：
```
增加内心独白的真实感，使用不完整的思绪，添加身体反应描写（心跳/手抖/呼吸），减少直白的情感陈述，用行为和细节暗示情感
```

**对话润色**：
```
增加口语化程度，添加语气词和停顿，为每个角色设计独特的说话方式，增加潜台词和言外之意，减少信息量过大的对话
```

### 风格增强技术

1. **词汇替换**：将通用词替换为更具画面感的词
2. **句式变换**：将陈述句变为反问/感叹/倒装
3. **细节增加**：在关键处添加感官细节
4. **节奏调整**：长短句交替，快慢节奏切换
5. **留白处理**：删除过度解释，留给读者想象
