# 润色指令模板库

## 通用去AI化

```
请对以下文本进行去AI化润色：
1. 减少过于完美的句式结构
2. 添加适当不规则表达
3. 使用个性化语言
4. 偶尔用口语化表达
5. 避免机械化段落结构
6. 保留核心信息和意图
7. 使用2-3成复合句
8. 掺入5%口语填充词
```

## 战斗场景

```
请润色以下战斗场景：
1. 增强动作描写的画面感
2. 使用短句增强节奏
3. 添加感官细节
4. 减少过度解释
5. 加入主角即时感受
6. 让动作本身说话
```

## 情感场景

```
请润色以下情感场景：
1. 增加内心独白的真实感
2. 使用不完整思绪
3. 添加身体反应描写
4. 减少直白情感陈述
5. 用行为和细节暗示情感
6. 增加犹豫和矛盾
```

## 对话场景

```
请润色以下对话：
1. 增加口语化程度
2. 添加语气词和停顿
3. 为角色设计独特说话方式
4. 增加潜台词
5. 减少信息量过大的对话
6. 让对话更自然流畅
```

## 环境描写

```
请润色以下环境描写：
1. 增加感官层次（五感）
2. 用具体细节替代抽象描述
3. 让环境暗示情绪
4. 减少静态描写，增加动态元素
5. 用比喻增强画面感
```
