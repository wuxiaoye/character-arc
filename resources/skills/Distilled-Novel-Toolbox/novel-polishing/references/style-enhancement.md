# 风格增强技术

## 1. 词汇替换

### 原则
- 将通用词替换为更具画面感的词
- 将抽象词替换为具体词
- 将书面词替换为口语词（适当）

### 示例
- "他很生气" → "他太阳穴突突直跳"
- "风景很美" → "远山如黛，碧水似镜"
- "她很害怕" → "她后背发凉，手指不自觉地攥紧衣角"

## 2. 句式变换

### 方法
- 陈述句→反问句："他不会来的" → "他会来？怎么可能。"
- 长句→短句："他在心中暗暗发誓一定要变得更强" → "他发誓。要变强。"
- 正常语序→倒装："一道剑光划破天际" → "划破天际的，是一道剑光。"

## 3. 细节增加

### 方法
- 在关键处添加感官细节
- 一个精准的细节胜过十句形容
- 细节要服务于情绪/剧情

## 4. 节奏调整

### 方法
- 长短句交替
- 快节奏用短句
- 慢节奏用长句
- 关键时刻用极短句

## 5. 留白处理

### 方法
- 删除过度解释
- 让读者自己推理
- 用暗示替代明说
- 结尾留有余韵
