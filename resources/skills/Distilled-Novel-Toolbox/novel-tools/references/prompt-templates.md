# 提示词模板库

## 大纲生成

```
你是一位专业的网文编辑。请根据以下信息生成一份详细的小说大纲：

题材：[题材类型]
核心设定：[金手指/核心设定]
主角人设：[性格标签+背景]
目标平台：[平台名称]
预计字数：[总字数]

要求：
1. 包含完整的起承转合
2. 每卷有明确的核心冲突
3. 标注爽点分布位置
4. 标注断章钩子位置
5. 字数分配合理
6. 每卷末有阶段性高潮
```

## 章节生成

```
你是一位网文写手。请根据以下信息撰写第[N]章：

前情提要：[前一章概要]
本章目标：[本章要完成的剧情目标]
出场角色：[角色名+当前状态]
爽点设计：[本章爽点类型+结构]
断章钩子：[本章结尾钩子类型]
文风要求：[文风类型]
字数要求：[字数范围]

要求：
1. 开头承接上一章
2. 节奏紧凑，不注水
3. 爽点结构完整（铺垫-爆发-余韵）
4. 结尾设置钩子
5. 角色行为符合人设
6. 口语化表达，避免AI味
```

## 角色对话

```
请为以下场景撰写对话：

角色A：[名字]+[性格]+[情绪]+[说话风格]
角色B：[名字]+[性格]+[情绪]+[说话风格]
场景：[场景描述]
对话目的：[推动什么剧情]
潜台词：[表面之下真正在说什么]

要求：
1. 每个角色有独特说话方式
2. 对话自然，不生硬
3. 包含潜台词
4. 推动剧情发展
5. 适当使用语气词和口头禅
```

## 润色去AI味

```
请对以下网文文本进行去AI化润色：

1. 减少完美句式结构
2. 添加不规则表达
3. 使用个性化语言
4. 偶尔用口语化表达
5. 避免机械化段落结构
6. 保留核心信息和意图
7. 使用2-3成复合句
8. 掺入5%口语填充词
9. 增加感官细节
10. 句长变化自然
```
