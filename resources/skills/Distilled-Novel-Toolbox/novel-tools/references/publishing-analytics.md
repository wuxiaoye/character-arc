# 发布与数据分析

## 关键数据指标

### 追读率
- 定义：读完上一章后继续读下一章的读者比例
- 重要性：最核心的数据指标
- 优化：断章钩子、节奏控制、爽点密度

### 完读率
- 定义：读完单章的读者比例
- 重要性：反映单章质量
- 优化：减少注水、加快节奏、增强钩子

### 收藏率
- 定义：阅读后收藏的读者比例
- 重要性：反映作品吸引力
- 优化：增强期待感、设定独特性、开篇质量

### 推荐票/月票
- 定义：读者主动投票支持
- 重要性：影响排名和推荐
- 优化：与读者互动、加更承诺、章节质量

## 数据优化策略

### 追读率低
- 原因：断章不够吸引/节奏拖沓/爽点不足
- 对策：加强断章钩子、加快节奏、增加爽点

### 完读率低
- 原因：章节注水/信息量不足/缺乏冲突
- 对策：精简内容、增加信息密度、每章有冲突

### 收藏率低
- 原因：设定不够吸引/开篇不够抓人/缺乏独特性
- 对策：增强设定吸引力、优化开篇、突出差异化

### 评论少
- 原因：缺乏讨论点/角色不够鲜明/剧情太平
- 对策：增加争议性话题、角色制造讨论、剧情增加反转
