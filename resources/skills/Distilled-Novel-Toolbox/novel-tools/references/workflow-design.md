# 工作流设计

## 标准工作流

### 第一阶段：构思（1-2天）

1. 确定题材方向
2. 确定核心设定/金手指
3. 确定主角人设
4. 确定目标平台
5. 生成初步大纲

### 第二阶段：大纲（3-5天）

1. 细化大纲至卷级
2. 细化大纲至章节级
3. 标注爽点与钩子
4. 建立设定文档
5. 设计主要角色

### 第三阶段：开篇（1周）

1. 精写黄金三章
2. 反复打磨开篇
3. 内投签约
4. 根据编辑反馈调整

### 第四阶段：连载（持续）

1. 日更2-3章
2. AI辅助生成初稿
3. 人工改写润色
4. 保持3-5天存稿
5. 关注数据反馈

### 第五阶段：运营（持续）

1. 分析数据指标
2. 与读者互动
3. 调整写作策略
4. 规划IP化路线

## 日更工作流

### 上午（3小时）
- 回顾前文和设定
- AI生成2章初稿
- 人工改写第1章

### 下午（3小时）
- 人工改写第2章
- 检查两章连贯性
- 发布/存稿

### 晚上（1小时）
- 回复读者评论
- 记录灵感
- 准备明天内容
