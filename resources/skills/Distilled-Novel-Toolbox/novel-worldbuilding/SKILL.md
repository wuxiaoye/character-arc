# 世界观与设定体系

## Purpose

提供网文世界观构建的完整方法论，从力量体系到地图格局，从经济体系到设定一致性维护，确保世界观宏大而自洽。

## When to Use

- 用户需要构建或完善世界观
- 用户需要设计力量/升级体系
- 用户需要规划地图与势力格局
- 用户需要建立经济/社会体系
- 用户遇到设定前后矛盾问题

## Instructions

### 力量体系架构

#### 修仙类模板
- 境界：炼气→筑基→金丹→元婴→化神→渡劫→大乘→飞升
- 每境界分：初期→中期→后期→巅峰
- 突破条件：灵力积累+悟道+天劫
- 战力跨度：每大境界10倍战力差

#### 玄幻类模板
- 等级：1-9阶或自定义命名
- 每阶分：初阶→中阶→高阶→巅峰
- 突破条件：修炼+机缘+战斗
- 特殊体系：异火/灵兽/血脉/天赋

#### 科幻类模板
- 文明等级：卡尔达肖夫I/II/III型
- 个人战力：基因等级/机甲等级/异能等级
- 科技树：材料→能源→信息→生物→空间

#### 都市类模板
- 异能等级：F→E→D→C→B→A→S→SS→SSS
- 组织架构：官方（异能局）+民间（世家/门派）
- 隐世规则：超凡世界对普通人隐藏

### 地图与势力格局

#### 地图扩展原则
1. 初期：小范围（村庄/学院/城市）
2. 中期：中范围（国家/大陆/星域）
3. 后期：大范围（世界/宇宙/多元）
4. 每次扩展都要有合理过渡

#### 势力设计模板
- 宗门/组织：3-5个主要势力+若干次要势力
- 势力关系：联盟/敌对/中立/暗斗
- 势力特色：每个势力有独特文化/功法/目标
- 势力平衡：不能一家独大，也不能完全均势

### 经济体系

#### 修仙经济
- 货币：灵石（下/中/上/极品）
- 资源：灵草/丹药/法器/功法
- 获取：任务/探索/炼制/交易
- 消耗：修炼/战斗/突破

#### 都市经济
- 货币：人民币+特殊货币
- 资源：异能材料/古董/灵物
- 获取：任务/商业/势力分配
- 消耗：修炼/装备/情报

### 设定一致性维护

#### 设定文档管理
1. 建立设定文档，记录所有已出场设定
2. 每次新增设定检查与旧设定是否矛盾
3. 定期回顾设定文档，清理冲突
4. 使用编号/标签系统管理设定

#### 常见设定崩坏
1. 战力崩坏：前期强敌后期弱鸡
2. 设定遗忘：前面提到的设定后面忘了
3. 逻辑矛盾：同一设定前后说法不同
4. 补丁过多：不断打补丁解释矛盾

### 悬念与谜题铺设

1. **世界级悬念**：世界的终极真相（贯穿全书）
2. **势力级悬念**：某势力的秘密（跨卷）
3. **人物级悬念**：某角色的真实身份/目的（跨章）
4. **事件级悬念**：某事件的真相（单卷/数章）
