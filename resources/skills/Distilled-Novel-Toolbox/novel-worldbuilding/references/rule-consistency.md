# 设定一致性维护

## 设定文档管理

### 必记内容
1. 力量体系：等级、突破条件、战力换算
2. 角色设定：性格、实力、关系、口头禅
3. 地图设定：地名、距离、特色、势力
4. 物品设定：名称、等级、效果、来源
5. 事件时间线：重要事件的时间顺序

### 管理方法
1. 使用表格/数据库管理设定
2. 每写完一章更新设定文档
3. 写新内容前查阅相关设定
4. 定期回顾清理冲突

## 常见设定崩坏

### 战力崩坏
- 表现：前期BOSS后期被杂兵秒杀
- 原因：战力膨胀失控
- 修复：设定战力上限/引入新维度

### 设定遗忘
- 表现：前面提到的设定后面再没出现
- 原因：设定太多记不住
- 修复：建立设定文档，定期回顾

### 逻辑矛盾
- 表现：同一设定前后说法不同
- 原因：写作时间跨度大，记忆偏差
- 修复：写前查阅，发现矛盾及时修正

### 补丁过多
- 表现：不断加设定解释之前的矛盾
- 原因：初期设定不严谨
- 修复：规划时多想几步，减少临时补丁
